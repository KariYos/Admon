ADMINISTRACIÓN

MANTENIMIENTO CLIENTES

Al ingresar al programa le aparecera un formulario para que usted pueda ingresar, eliminar o editar los datos del cliente que usted desee.


Para insertar datos primero presione el botón de ingresar del navegador, para preparar la base de datos. Luego ingrese los datos del cliente y luego presione el botón de guardar. Si lo que desea es editar el documento, presione la flechas para moverse entre los datos de los clientes que se ingresaron anteriormente. Después modifique el campo deseado y oprima el botón de guardar. Finalmente si lo que desea es eliminar al cliente, solamente utilice las flehas para moverse entre los registros, buscando el cliente que desea eliminar y presione el botón eliminar.
